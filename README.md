# docker_express_environment

docker と express 環境のボイラープレートを作りたい

## 詳細

- Express（アプリケーション側）

node は 16 系を使用、localhost の port を 4000 に変更

<br>

- mysql

特筆した変更はなし

<br>

## get started

```bash
make up
```
